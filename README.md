# react-todolist

## 설명

이 프로젝트는 React를 이용한 투두 리스트(To-Do List) 프로젝트입니다.

&nbsp;

## 목표

- 할 일(Todo)을 입력받아 리스트로 나열합니다.
- 버튼 클릭 시 할 일(Todo)를 삭제합니다.
