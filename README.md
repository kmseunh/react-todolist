# react-todolist

## 설명

이 프로젝트는 React를 이용한 To-Do 리스트 앱입니다.

&nbsp;

## 목표

- 할 일을 추가, 편집, 삭제할 수 있는 투두 리스트를 구현합니다.
- `Tailwind CSS`를 사용하여 빠르고 간편하게 스타일링합니다.
